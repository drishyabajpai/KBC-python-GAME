# 🎮 Python KBC Game

A beginner-friendly Kaun Banega Crorepati (KBC) game built in Python.

## Features

- 20 Questions
- Prize Money System
- Random Question Order
- Replay Option
- Correct Answer Explanation
- Sound Beep

## Technologies

- Python
- Random Module
- Winsound

## Run

```bash
python main.py
```
